# 网络搜索

通过 `web_search` 搜索互联网，`web_fetch` 获取网页内容。

## 可用工具

| 工具 | 用途 |
|------|------|
| `web_search` | 搜索互联网信息 |
| `web_fetch` | 获取网页正文内容 |

## 使用场景

- 查找某个概念/技术的介绍
- 获取某个网站/文档的内容
- 搜索最新资讯/论文/教程
