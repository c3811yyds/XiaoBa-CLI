# 多智能体协作

通过 `task` 工具向其他智能体分发任务，`task_output` 获取结果，`task_stop` 停止任务。

## 可用工具

| 工具 | 用途 |
|------|------|
| `task` | 向其他智能体分发任务 |
| `task_output` | 获取任务执行结果 |
| `task_stop` | 停止正在执行的任务 |
