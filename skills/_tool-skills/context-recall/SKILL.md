# 上下文回忆

查询历史操作记录，回忆之前的工具调用和产出文件。

## 可用工具

| 工具 | 用途 |
|------|------|
| `recall_log` | 查询历史操作记录 |

## 使用场景

- 用户提到之前做过的事情（如"刚才那个论文"、"上次分析的结果"）
- 需要查看之前某轮对话中使用了哪些工具、产出了哪些文件
- 用户要求基于之前的操作继续工作

## recall_log 用法

```json
{"query": "<关键词>", "turns_back": 5}
```

参数说明：
- `query`（可选）：搜索关键词，匹配任务描述、工具名、文件路径等
- `turns_back`（可选）：回溯几轮对话记录（默认 5，最大 20）

## 示例

查询最近 5 轮操作：
```json
{}
```

搜索与"论文"相关的操作：
```json
{"query": "论文"}
```

查询最近 10 轮操作：
```json
{"turns_back": 10}
```
