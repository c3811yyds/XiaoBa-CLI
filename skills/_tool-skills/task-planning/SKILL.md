# 任务规划

通过 `task_planner` 拆分复杂任务为可执行步骤，用 `todo_write` 管理 TODO 列表。

## 可用工具

| 工具 | 用途 |
|------|------|
| `task_planner` | 拆分复杂任务为步骤 |
| `todo_write` | 管理 TODO 列表 |
| `enter_plan_mode` | 进入计划模式 |
| `exit_plan_mode` | 退出计划模式 |
