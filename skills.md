# XiaoBa Skills

## 可用技能列表

1. **agent-browser** - 浏览器自动化工具，用于与网页交互、导航、填写表单、点击按钮、截图等

2. **opencli** - 命令行操作社交平台（B站、知乎、Twitter、YouTube、微博、小红书、Reddit、HackerNews、雪球等）

3. **report** - 生成每日工作报告、日报、工作总结

4. **self-evolution** - 自我进化：引导创建新的 skill 或 Python tool，扩展能力边界

5. **skill-publish** - 发布 Skill 到官方 Skill Hub

6. **sub-agent** - 子智能体管理：派遣、查看、停止子智能体执行耗时长的任务
