Important Details

这轮产品化里已经定下的边界

  • Dashboard 不是隐式配置源，只能受控查看、预览、校验和编辑明确 schema。
  • secret 不进入 Runtime Profile；settings API 只展示 presence，不回显值或 suffix。
  • CatsCo 托管模型是默认产品方向；当前本地只显示边界，不伪装云端模型已经接入。
  • buildsense.asia 是历史/内部网关，不作为普通用户配置入口。
  • 自定义模型保留为 Advanced 路径，当前落到 GAUZ_LLM_*。
  • 备用模型已清理：UI/API/runtime 都不再把 GAUZ_LLM_BACKUP_* 当产品能力读取。
  • AgentSession 不重新变胖；adapter 继续保持通讯层定位。
  • Skill Hub 和 Companion 是同一产品域，但写操作只属于 Skill Hub。
